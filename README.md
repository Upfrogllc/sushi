# Sushi Sportfishing — sushisf.com

The official website for Sushi Sportfishing — Captain Charley Pereira's world-champion offshore charter operation based at Pirate's Cove Marina, Manteo, NC.

## Deploy

1. Push this repo to GitHub
2. Connect to Netlify (publish directory: `.`)
3. Every push to `main` auto-deploys

## Pages

- `/` — Homepage
- `/the-team/` — Captain Charley's full story
- `/the-boat/` — 57' IBW specs & photos
- `/tournaments/` — Championship record
- `/charters/` — Charter types & booking
- `/gallery/` — Photo gallery
- `/contact/` — Booking form & contact info

## Contact Form

Uses Netlify Forms (built-in, free). The form on `/contact/` has `data-netlify="true"` — submissions go to your Netlify dashboard.

## Custom Domain

Set `sushisf.com` in Netlify → Domain Settings → update DNS nameservers at your registrar.
